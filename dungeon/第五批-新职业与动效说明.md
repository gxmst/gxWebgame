## 1. 战斗内核扩展(本批的地基)

前三批明确推迟召唤流,因为它把战斗从"1 vs 多"变成"多 vs 多"。本批正式落地,新增两类机制:

### 1.1 友方随从(summon)
- 玩家侧新增 `minions` 单位数组:战斗中由召唤技能创建,按速度进入回合序;
  **本回合召唤的随从下一轮才行动**(召唤病),同种子完全可复现。
- **敌方目标选择**:敌人在"玩家 + 存活随从"里选最低血目标——骷髅天然成为肉盾。
- 随从属性 = 施法时施法者当前属性 × 技能比例(升级可提升比例/上限/每次召唤数)。
- 场上随从数受双重上限:技能 `maxMinions` 与内核硬上限 `combat.minions.maxActive`(5)。
- 玩家阵亡即判负(随从不续战);随从阵亡记入战报,冷却好后 AI 自动补召。
- 快照/结算全链路携带 `minions`,战报新增"召唤物伤害""召唤/折损"。

### 1.2 形态增益(empower)
- 新技能类型:持续 N 回合的**主动输出增益**(增伤 % + 附加吸血 %),
  区别于 guard(减伤/闪避)——由施法 AI 主动保持覆盖(没有增益时优先开启)。
- 结算位置:攻方 `empower.damageBonus` 乘进攻击力;`lifestealBonus` 并入吸血(共享吸血上限)。

### 1.3 施法 AI
优先级扩展为 `survival → summon → empower → aoe → single → basic`(可配置,旧配置自动补全)。

## 2. 新职业

| | 死灵法师 ⚰️ | 德鲁伊 🐻 |
|---|---|---|
| 定位 | 亡灵召唤(智力/法术系) | 形态搏杀(力量+体质/物理系) |
| 基础 | 骨矢 🦴 | 利爪横扫 🐾 |
| 单体 | 骨矛 🗡️(1.7x) | 野性撕咬 🐺(1.65x) |
| 群体 | 尸爆 ☣️(0.78x) | 自然之怒 ⛈️(0.74x) |
| 特色 | 亡者复生 💀:召 2 名骷髅战士(上限 3,满级 4/次 3),属性随施法者成长 | 狼魂形态 🌕:+32% 伤害 +10% 吸血持续 3 回合(满级 +50%);回春术 🌱:低血回 30% |
| 打法 | 开局铺骷髅承伤,本体尸爆/骨矛收割 | 开形态贴脸互殴,吸血+回春打消耗战 |

- 新增 `damageType`(magic/physical)职业字段:法师/死灵吃"法术增伤"词条,战/游/德吃"物理增伤";词条池 classIds 同步。
- 德鲁伊攻击同时吃力量和体质(独有),天生耐揍;死灵本体脆但骷髅是可再生的盾。
- 平衡模拟(40 场/点):两职业普通层推进 98–100% 胜率、Boss 层"认真构筑"全过,与战士基线一致;死灵场均回合最短(4 左右),德鲁伊最长(8–10,消耗流手感)。

## 3. 前端动效(全部尊重 prefers-reduced-motion)

- **战斗飘字**:伤害(红)/暴击(橙、加大)/闪避(灰)/治疗(绿)/燃烧(橙),从日志事件驱动,飘起渐隐;跳过战斗时不会刷屏。
- **受击闪烁**:被命中的卡片短促提亮。
- **随从棋子**:英雄卡下方一排小棋子(emoji + 微型血条),召唤时弹入,阵亡置灰;悬停显示名字与血量。
- **日志逐条滑入**、**血条平滑过渡**(原有 transition 保留)、**阵亡敌人灰化**。
- **弹窗/遮罩/Toast 入场动画**;结算内容(横幅、收益、战利品卡、战报)依次升起,多件战利品错峰入场;升级提示脉冲。
- 系统开启"减少动态效果"时:JS 直接不生成飘字,CSS 全局动画时长压到 0.01ms。
